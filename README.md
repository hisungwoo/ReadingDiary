<h2>:memo: 상세 내용 </h1>

* 달력에 독서날 스티커를 추가하여 내가 얼마나 독서를 하였는지 쉽게 볼 수 있습니다.
 
* 책을 검색해서 간편하게 추가가 가능합니다.
 
* 해당 책의 독서 진행률을 볼 수 있습니다.



<br>
<h2>:computer: 개발 내용 </h1>

* 안드로이드 앱 전체 기획 및 개발

* Jetpack 기능 활용하여 MVVM 패턴 적용

* Coroutine을 활용한 비동기, 동시성 프로그래밍

* Retrofit을 활용한 API 연동으로 검색 기능 개발

* DataBinding을 이용한 실시간 UI 변경 및 소스 간결화

* Room를 이용한 데이터베이스 활용

* Jsoup를 이용한 웹 크롤링

* 앱 출시 예정
 
* 출시 후에도 공부한 기술 스택 적용 후 지속적인 업데이트
